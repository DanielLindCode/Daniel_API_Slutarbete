Daniels API

This API as four functions.

  • Calculator
  • Rock, Paper, Scissor Game
  • Random image display
  • Csv file reader

Read more about the AIP groups bellow.

Calculator group

The calculator takes in three parameters. An operator, number1 and number2. This calculator has operators PLUS, MINUS, TIMES, DIVIDE.

The root "/calc" displays information about the group.

[GET] /calc

Example

Plus operator example

[GET] /calc/PLUS/5/10

Result

Minus operator example

[GET] /calc/MINUS/100/25

Result

Divide operator example

[GET] /calc/DIVIDE/40/6

Result

Times operator example

Rock, Paper, Scissor -Game group

The rock, paper, scissors game takes in one parameter (playerMove). The moves you can make is "rock", "paper" and "scissors".

The root "/rsp" displays information about the group.

[GET] /rps

Example

Play against the computer!

[GET] /rps/[playerMove]

Play Paper!
Play Scissors!
